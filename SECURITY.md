# Politique de sécurité / Security Policy

## 🇫🇷 Signaler une vulnérabilité
Si tu découvres une faille de sécurité, **n'ouvre pas d'issue publique**.
Contacte-nous en privé via le **Discord** : <https://discord.gg/BmUHBf2bnm>
(ou par les *Security advisories* GitHub du dépôt). Nous répondrons au plus vite.

## 🇬🇧 Reporting a vulnerability
If you find a security issue, **please do not open a public issue**. Report it
privately via **Discord** (<https://discord.gg/BmUHBf2bnm>) or through the
repository's GitHub *Security advisories*.

## Portée / Scope
Cette application est une coquille Electron qui charge la carte de
`https://33immortals.fr`. Elle ne stocke ni identifiants ni données sensibles en
local ; les préférences (raccourcis, etc.) restent sur la machine de l'utilisateur.
